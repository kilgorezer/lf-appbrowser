THIS HAS BEEN ONLY TESTED ON THE LATEST LEAPPAD PLATINUM FIRMWARE VERSION

1: Using LFManager execute this command (or one with an equivelent function) to enable FTP: /etc/init.d/vsftpd start &
2: Using any FTP software that allows uploading folders, upload the Kilgorezer folder to your tablet.
3: Find your tablet's local device IP, run init.py, and enter the IP into it.
4: Using LFManager execute this exact command ONCE: /bin/sh /Kilgorezer/init
5: Restart the device.
